## 遊戲設計（V13 現況）

### 基本規則

| 項目 | 內容 |
|---|---|
| 登入 | 輸入序號（如：001），進入說明頁確認後才開始計時 |
| 時限 | 20分鐘 |
| 地圖 | Zelda 地下城風格，9個房間 3×3，每次顯示一個 5×5 房間，滑入特效換房 |
| 玩家 | 🏃 跑步動畫，鍵盤方向鍵／WASD 或觸控滑動移動 |
| 火焰 | 🔥 emoji，踩上觸發題目；答完熄滅，20秒後重生並換新題 |
| HP | 10滴血，答錯扣1滴，歸零3秒倒數後復活回大禮堂（血量全滿） |

### 遊戲功能（V11–V13 累積）

| 功能 | 說明 |
|---|---|
| +1浮動文字 | 答對後金黃色「+1」從火焰位置飄起 |
| 爆炸粒子特效 | 12顆彩色粒子爆散，每顆動態注入獨立 keyframes（修正CSS變數在keyframes不支援問題） |
| 連擊數 | 連續答對顯示「🔥 連擊 ×N」，答錯歸零 |

| 每輪結束視窗 | 答滿40題後跳出統計，可選繼續練習或結束 |
| 鼓勵訊息 | 每5/10/15分鐘自動顯示激勵文字 |
| 斷線恢復 | 每5秒 localStorage 存檔，重整後自動恢復進度 |

## 研究數據（Google Sheets 15欄）

| 類別 | 欄位名稱 | 說明 |
|---|---|---|
| 基礎識別 | Timestamp | 台灣時間（Asia/Taipei） |
| | UserName | 受試者序號 |
| | SessionID | 本次遊戲唯一識別碼 |
| 研究核心 | QuestionID | 題目編號 |
| | BloomLevel | 認知層次（L1–L5） |
| | QuestionCategory | 前測／後測／共通／題庫 |
| | ResponseTime_ms | 作答時間（毫秒） |
| | IsCorrect | 是否答對（true/false） |
| | ChosenOption | 選擇選項（A/B/C/D） |
| | FirstAttemptCorrect | 本題第一次嘗試是否正確 |
| | AttemptNum | 本題累計嘗試次數（跨輪累計） |
| 遊戲行為 | RoomName | 作答時所在房間 |
| | HPAtAnswer | 作答時血量 |
| | DeathCount | 累計死亡次數 |
| | TotalPlayTime | 累計遊玩秒數 |

遊戲結束時另送一筆 `QuestionID = SESSION_END`，記錄完整遊玩時間與最終狀態。

## 版本演進

| 版本 | 日期 | 重點 |
|---|---|---|
| **V1** | 1150425 | 初版建立。單一 Canvas 大地圖（23×14格），火焰隨機散布全圖，踩上觸發題目。HUD 顯示「火焰已滅數」、「累計答對」、「學習時間（正計時）」與進度條。底部有 Bloom 層次與題目分類圖例。Sheets URL 預留空白（未接通）。無登入頁、無計時限制、無HP系統。 |
| **V2** | 1150425 | 新增登入頁（輸入姓名才能進入）、火焰飄落動畫背景。HUD 加入玩家名稱顯示。地圖維持單一大地圖但自適應畫面寬度。送出資料欄位初步建立（sessionId、userId、timestamp、questionId、bloom、category、isCorrect 等）。Sheets URL 仍為空白。 |
| **V3** | 1150425 | 地圖改為具有走廊結構的衛生局平面圖（3層樓、3條走廊、各科室獨立方塊，21×19格）。新增 20分鐘倒數計時器（計時從登入後開始）。每題新增答題計時（秒數顯示）。HUD 改為「剩餘時間」、「累計答對」、「作答次數」。火焰熄滅後重生時間定為10秒。Sheets 欄位沿用 V2。 |
| **V3 final** | 1150425 | 僅修正一處：火焰熄滅後重生時間從10秒改為20秒（配合實際學習節奏調整）。其餘與 V3 完全相同。 |
| **V4** | 1150425 | 地圖改以 RAW_MAP 字元陣列定義（13×22格），精確還原衛生局各樓平面格局（局長室、企劃科、電腦機房、行政科、醫政科、大禮堂、疾管科等），並加入走廊與門格。登入頁改版（UI優化）。新增遊戲結束畫面（顯示答題統計）。計時改為說明提示後才開始。Sheets 欄位新增 `responseTime_ms`、`chosenOption`、`correctOption`、`remainingTime`。無HP系統、無多房間切換。 |
| **V5** | 1150425 | 地圖架構全面改為 **Zelda 地下城風格**，9個獨立房間（3×3配置），每次只顯示一個 5×5 房間，走到門口箭頭滑入切換房間。房間清單確立（局長室、企劃科、電腦機房、行政科、大禮堂、醫政科、檢驗科、疫苗室、疾管科）。加入各房間家具繪製。新增房間名稱橫幅（room-banner）。Sheets URL 仍空白，欄位與 V4 相同。無HP系統。 |
| **V6** | 1150425 | 新增 HP 系統（10滴血❤️，答錯扣1滴，歸零後復活）。Sheets 欄位擴充至完整15欄，新增 `SessionID`（遊戲唯一識別碼）、`RoomName`、`HPAtAnswer`、`DeathCount`、`TotalPlayTime`、`FirstAttemptCorrect`、`AttemptNum`，並接上真實 Apps Script URL 開始傳送數據。 |
| **V7** | 1150426 | 新增遊戲說明頁（確認後才開始計時）。火焰從 Canvas 繪製圓形改為 🔥 Emoji 顯示，視覺更直觀。登入頁火焰粒子動畫效果優化。 |
| **V8** | 1150428 | 遊戲結束時額外送出一筆 `SESSION_END` 紀錄，TotalPlayTime 記錄實際遊玩秒數（玩滿20分鐘=1200）。`AttemptNum` 改為跨輪累計（整個 session 內同一題的嘗試次數累加）。 |
| **V9** | 1150428 | 移除題目顯示中的 Bloom 層次與題目分類標籤（防止受試者在答題時看到分類資訊而產生提示效果）。答題解說文字優化，正確/錯誤回饋更清楚。 |
| **V10** | 1150503 | 修正 Google Sheets 資料傳送穩定性問題（Content-Type 改為 text/plain 解決 CORS 預檢問題）。通過壓力測試（多人同時送出不遺漏）。 |
| **V11** | 1150508 | 新增 localStorage 存檔機制，每5秒自動存檔，重整頁面後自動恢復進度（房間位置、HP、答題紀錄、計時）。存檔超過25分鐘自動作廢。 |
| **V12** | 1150509 | 登入改為輸入序號（原為姓名）。Timestamp 改為台灣時間（Asia/Taipei）。玩家圖示改為 🏃 跑步動畫（上下彈跳＋傾斜）。答對特效（+1浮動文字、爆炸粒子、連擊數）改為按「繼續探索」後延遲350ms觸發。爆炸粒子改用動態注入獨立 keyframes，修正 CSS 變數在 keyframes 中不支援導致粒子不動的問題。 |
| **V13** | 1150511 | 題庫全面替換為老師修正版40題。語氣統一改為「本局」。選項文字完整化。刪除「延長界限時間」題，補入「管理權人定義」新題（id:32）。Bloom 層次重新對應（火警探測器誤動作升為L5評鑑）。 |
