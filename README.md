# fire-drill-game## 版本演進

**V1 — 基礎 Demo**（1150425）
Canvas 遊戲雛形，WASD移動，踩火焰觸發答題，20題示範題庫，下方顯示研究數據紀錄表。

**V2 — 建築物平面圖 + Google Sheets串接**（1150425）
加入建築物平面圖（多層樓格局），40題完整題庫，串接Google Sheets，支援手機觸控滑動，登入頁輸入姓名。

**V3 — Zelda地下城風格地圖 + 20分鐘倒數**（1150425）
改為多房間地圖（3層樓9個房間），火焰20秒重生，無限循環練習，20分鐘倒數計時，結束後顯示成績摘要。

**V4 — 手機直屏優化 + 高對比配色**（1150425）
針對手機直屏重新設計，土黃色木紋地板、灰磚牆壁，單層平面圖，移除多層樓設計，火焰密集分布。

**V5 — 單房間Zelda視角 + 滑入換房特效**（1150425）
每次只顯示一個5×5房間，走到門口箭頭觸發房間切換，畫面整體滑入動畫，9個房間3×3連通配置。

**V6 — HP系統 + 家具圖示 + 完整15欄數據**（1150425）
加入HP血量系統（10滴血、答錯扣1滴、歸零3秒後復活回大禮堂），各科室主題家具圖示，完整15個研究數據欄位，死亡次數列入數據記錄。

**V7 — 說明頁 + 🔥 Emoji火焰 + 計時修正**（1150426）
加入遊戲說明頁（輸入姓名後先看說明，按確認才開始計時），遊戲內火焰改為🔥 Emoji動畫版，拿掉底部操作說明文字，Bloom層次標籤移除。

**V8 — 後台遊戲時間修正 + AttemptNum修正**（1150428）
遊戲結束時送出SESSION_END完整遊玩時間紀錄（玩滿20分鐘=1200秒），AttemptNum復活後不重置，改為整個session累計。

**V9 — 答題面板優化**（1150428）
移除答題面板的Bloom層次與前測/後測分類標籤，避免洩題影響實驗結果。解說文字放大至16px，答對綠底白字、答錯紅底白字。

**V10 — 資料傳送穩定性修正**（1150503）
Content-Type改為text/plain，解決CORS問題，確保答題數據穩定送達Google Sheets。通過80人持續20分鐘壓力測試。

**V11 — 留存機制全面升級 + 斷線恢復**（1150508）
答對顯示+1浮動文字與粒子爆炸特效，連續答對顯示連擊數，均於按「繼續探索」後觸發。HUD新增累計答題數。每累計答完40題跳出選擇視窗（繼續練習或結束學習）。第5、10、15分鐘彈出鼓勵訊息。新增localStorage斷線恢復，每5秒自動存檔，重整頁面自動恢復遊戲狀態。
