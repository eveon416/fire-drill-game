## 遊戲設計（V14 現況）

### 核心概念

橫向捲軸RPG戰鬥與防災答題結合。主角固定在畫面左側，背景三層視差捲動製造前進感，怪物從右側出現並走向主角，遭遇後停下答題，答題結果直接驅動戰鬥動畫。

### 畫面佈局

- 上方 1/3：HUD資訊列 + 戰鬥場景（Canvas）
- 下方 2/3：防災答題區（可捲動）
- 全螢幕比例配置，手機、平板、電腦均適配，不設固定px限制

### 戰鬥場景（Canvas 繪製）

**背景（三層視差捲動）**

- 遠景：建築剪影 + 星點，速度最慢（速率0.3）
- 中景：地板磚縫，中速（速率0.8）
- 近景：木柵欄，速度最快（速率1.4）
- 遭遇怪物後背景停止捲動，等待答題

**主角（Canvas 繪製 Q版小人）**

- 大圓頭、圓潤橢圓身體、腮紅、大瞳孔，Q版可愛風格
- 跑步時有彈跳動畫與身體傾斜
- 受擊時眼睛變×、嘴巴拉平、往後退
- 攻擊時身體前傾、眼睛張大
- 帽子依升階不同（見升階系統）

**怪物（Canvas 繪製 Q版圓滾滾）**

- 一般怪：圓滾滾造型，光暈漸層色，待機搖擺動畫，小腳
- Boss：體型放大1.6–2.0倍、旋轉刺冠、持續旋轉光圈特效

**HUD資訊列**

- 玩家序號、機關名稱
- 剩餘時間（最後60秒變紅閃爍）
- 正確率
- 累計已答題數
- HP心型列（10顆，答錯變灰）
- 關卡進度條（STAGE N，每10關一段）

### 答題流程

1. 怪物從右側走向主角，到達停下
2. 下方題目區出現題目與四個選項（A/B/C/D）
3. 玩家點選答案
4. **不管答對或答錯，都立即顯示正確答案與解析說明**
5. 玩家按「繼續」按鈕才推進到下一題

**答對（一般怪）**
- 主角做攻擊動畫（前傾衝刺）
- 怪物爆炸成粒子消失
- 金黃色 +1 浮動文字從場景飄起
- 如有連擊則顯示「🔥 連擊 ×N」

**答錯（一般怪）**
- 怪物做衝刺動畫撞向主角（捨身攻擊）
- 主角受擊動畫（×眼、往後退、畫面震動）
- 怪物消失
- 主角扣1滴血

**答對（Boss，未滿3擊）**
- 主角攻擊動畫
- Boss血條減少
- 顯示「還需N擊」，玩家按繼續後出下一題

**答對（Boss第3擊）**
- Boss爆炸粒子消失
- 大量特效爆發
- 通關，進入下一關

**答錯（Boss）**
- Boss做衝刺動畫撞向主角後**退回原位**（不消失）
- 主角受擊動畫
- 主角扣1滴血
- Boss退回後顯示「繼續挑戰Boss（還需N擊）→」按鈕
- 玩家按繼續才出下一題

### 關卡系統

- 每答對一題通過一關，關卡數累計不重置
- 每10關出現一隻Boss
- Boss需答對3題才能擊倒（答錯扣血但不扣Boss進度條）
- 答題標記為用過（40題一輪不重複），答錯也算用過
- 第40題答完後跳出每輪結算視窗（繼續練習 / 結束學習），不論當時是否在打Boss中途
- 若Boss戰進行到第40題，結算後Boss進度保留，新一輪開始後可接續挑戰

### 主角升階系統

每次通過Boss關（打倒Boss），若累計關卡數達到門檻，主角自動升階，帽子外觀改變，並跳出升階通知視窗：

| 累計關卡 | 外觀 | 稱號 |
|---|---|---|
| 第1關起 | 黑色頭髮 | 一般員工 |
| 第11關起 | 黃色安全帽 | 初級防災員 |
| 第21關起 | 深色帽＋橘色反光條 | 防災專員 |
| 第31關起 | 紅色消防頭盔＋金徽章 | 消防員 |
| 第41關起 | 深紅金徽消防帽＋金星 | 防災達人 |

### Boss特效疊加系統

每打倒一個Boss，主角身上**永久疊加**一層特效，越打越強，視覺越來越華麗：

| 累計打倒Boss數 | 疊加特效 |
|---|---|
| 1 | 6顆金色★繞主角旋轉 |
| 2 | 再加藍紫雙光圈脈衝（相位不同） |
| 3 | 再加5個🔥火焰飄散頭頂周圍 |
| 4以上 | 再加閃電弧光在身體周圍隨機跳動 |

### HP系統

- 初始10滴血，以❤️心型圖示顯示
- 答錯扣1滴血，同時畫面震動特效
- 血量歸零：出現死亡覆蓋畫面，倒數3秒後滿血繼續
- 復活後從當前關卡繼續，不重置任何進度

### 計時與特殊訊息

- 說明頁按「出發冒險！」後才開始計時，共20分鐘
- 計時顯示在HUD右上角，最後60秒變紅色閃爍
- 第5、10、15分鐘時螢幕彈出鼓勵訊息
- 時間到顯示結束畫面（詳見下方）

### 結束畫面

顯示以下統計資訊：

- 作答次數（含重複題）
- 答對次數
- 正確率
- 涵蓋題數（不重複）/ 40
- 死亡次數
- 最高連擊數
- 依正確率顯示不同鼓勵訊息（≥90% / ≥70% / ≥50% / 以下）

## 研究數據欄位（Google Sheets，15欄）

每次答題即時送出一筆資料：

| 類別 | 欄位名稱 | 說明 |
|---|---|---|
| 基礎識別 | Timestamp | 台灣時間（Asia/Taipei） |
| | UserName | 玩家輸入序號（如：001） |
| | SessionID | 本次遊戲唯一識別碼（S+時間戳） |
| 研究核心 | QuestionID | 題目ID |
| | BloomLevel | 認知層次（L1–L5） |
| | QuestionCategory | 題目分類（前測/後測/共通/題庫） |
| | ResponseTime_ms | 答題時間（毫秒） |
| | IsCorrect | 是否答對（true/false） |
| | ChosenOption | 選擇的選項（A/B/C/D） |
| | FirstAttemptCorrect | 是否第一次嘗試即答對（true/false） |
| | AttemptNum | 本題在此session的累計嘗試次數 |
| 遊戲行為 | RoomName | 作答時所在關卡（stage_N） |
| | HPAtAnswer | 作答時的HP值 |
| | DeathCount | 累計死亡次數 |
| | TotalPlayTime | 累計遊玩秒數 |

遊戲結束時另送一筆 `QuestionID = SESSION_END`，記錄最終HP、死亡次數與實際遊玩總秒數（玩滿20分鐘 = 1200）。
