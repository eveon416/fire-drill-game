# fire-drill-game
防災演練學習系統
V1 — 基礎 Demo
Canvas 遊戲雛形，WASD移動，踩火焰觸發答題，20題示範題庫，下方顯示研究數據紀錄表。
V2 — 建築物平面圖 + Google Sheets串接
加入建築物平面圖（多層樓格局），40題完整題庫，串接Google Sheets，支援手機觸控滑動，登入頁輸入姓名。
V3 — Zelda地下城風格地圖 + 20分鐘倒數
改為多房間地圖（3層樓9個房間），火焰20秒重生，無限循環練習，20分鐘倒數計時，結束後顯示成績摘要。
V4 — 手機直屏優化 + 高對比配色
針對手機直屏重新設計，土黃色木紋地板、灰磚牆壁，單層平面圖，移除多層樓設計，火焰密集分布。
V5 — 單房間Zelda視角 + 滑入換房特效
每次只顯示一個5×5房間，走到門口箭頭觸發房間切換，畫面整體滑入動畫，9個房間3×3連通配置。
V6 — HP系統 + 家具圖示 + 完整15欄數據
加入HP血量系統（10滴血、答錯扣1滴、歸零3秒後復活回大禮堂），各科室主題家具圖示，完整15個研究數據欄位，死亡次數列入數據記錄。
V7 — 說明頁 + 🔥 Emoji火焰 + 計時修正
加入遊戲說明頁（輸入姓名後先看說明，按確認才開始計時），遊戲內火焰改為🔥 Emoji動畫版，拿掉底部操作說明文字，Bloom層次標籤移除。
V8 — 後台遊戲時間修正 + AttemptNum修正
遊戲結束時送出SESSION_END完整遊玩時間紀錄（玩滿20分鐘=1200秒），AttemptNum復活後不重置改為整個session累計。
V9 — 移除答題面板的Bloom層次和前測/後測分類標籤，避免玩家看到題目分類資訊影響實驗結果。
解說文字從13px放大至16px，答對改為綠底白字、答錯改為紅底白字，提升閱讀對比度。
